# World Biomes (The "Where")

Every quest happens *somewhere*. The biome dictates the hidden difficulty multipliers.

## 1. The Temperate Lands (Starting Zone)
| Biome | Description | Hidden Multiplier | Required Tag | Hazard |
| :--- | :--- | :--- | :--- | :--- |
| **Grassy Plains** | Open fields, safe roads. | 1.0x (Standard) | None | Rain (Rusts armor slightly). |
| **Deep Forest** | Dense trees, limited vision. | 1.2x Difficulty | `Tracking` or `Ranger` | **Ambush**: Party starts surrounded. |
| **Old Ruins** | Crumbling stone towers. | 1.5x Trap Chance | `Dungeon Delver` (Rogue) | **Collapse**: Heavy Armor risks falling damage. |

## 2. The Hazards (Mid-Game)
| Biome | Description | Hidden Multiplier | Required Tag | Hazard |
| :--- | :--- | :--- | :--- | :--- |
| **The Poison Swamp** | Stinking bog, purple mist. | 2.0x vs Low VIT | `Poison Resist` | **Disease**: Adventurers return sick (Unavailable for 3 days). |
| **Frozen Peaks** | Snow, ice, howling wind. | 2.0x vs Low STR | `Cold Resist` or `Fur` | **Frostbite**: Reduces DEX permanently if failed. |
| **Scorched Desert** | Endless sand, water is scarce. | 1.5x Stamina Drain | `Water Skin` item | **Heatstroke**: Heavy Armor wearers pass out (Auto-Fail). |

## 3. The Death Zones (End-Game)
| Biome | Description | Hidden Multiplier | Required Tag | Hazard |
| :--- | :--- | :--- | :--- | :--- |
| **Active Volcano** | Lava rivers, ash in air. | 3.0x Damage | `Fire Immune` | **Magma**: Instant Death for anyone who trips. |
| **The Void-Pit** | Gravity is wrong. Whispers. | 5.0x Sanity Drain | `Iron Will` | **Madness**: Adventurer betrays the guild upon return. |
| **Underwater Ruins** | Sunken city. | Special (Requires Magic) | `Water Breathing` | **Drowning**: Instant death without magic. |
